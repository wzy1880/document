# 文章列表

需要token:
`不需要`

请求地址:
`/weber-api/resource/articles`

请求方式:
`get`

请求参数:

| 参数  | 必填  | 类型 | 说明 |
| ---- | ----  | ---- | ---- |
| title |  否  | string | 搜索标题 |

状态码:`200`
